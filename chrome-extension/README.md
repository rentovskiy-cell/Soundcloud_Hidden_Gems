# SoundCloud Hidden Gems — Chrome Extension

Load this folder in Chrome: `chrome://extensions` → Developer mode → **Load unpacked**.

Full docs: [../README.md](../README.md)

## v2.0.0 highlights

- Filter presets, dig session, playback, My Gems & Session export
- Premiere channel artist parsing
- Compact stats, onboarding, icons

## Before Chrome Web Store

1. Replace `YOUR_USERNAME` in `sidepanel.js` (`PRIVACY_POLICY_URL`) and `docs/`
2. Enable GitHub Pages on `/docs`
3. Take 3–5 side panel screenshots
